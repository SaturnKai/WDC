On a more serious note, yesterday I was able to connect a door / window contact sensor to my garage door, and utilize an ESP32 with a 433MHz RF receiver to receive the binary data sent from the transmitter when my garage door is opened or closed. Once the state of the sensor changes, it sends an HTTP post request data to my webserver, which then sends a push notifcation to my phone using the Pushover API.
